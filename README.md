# Tactical Path Game

A tactical planning game built with python and pygame.

## 🎮 Idea
Players draw attack paths during a planning phase. After time runs out, units move and battle automatically.

## 🚀 Features (Planned)
- Path drawing system
- Planning timer
- Unit movement
- Combat system
- Win conditions

## 🛠️ Tech Stack
- Python
- Pygame

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
